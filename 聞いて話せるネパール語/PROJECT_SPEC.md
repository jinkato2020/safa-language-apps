# 「聞いて話せる○○語」アプリ仕様書

このドキュメントは、日本人向け語学学習アプリのコンセプト・実装仕様を完全に記述したものです。別言語版（例: 英語、ヒンディー語、韓国語）を作る際の**完全な複製ガイド**として使用できます。

---

## 1. コンセプト

### キャッチコピー
**「聞いて話せる○○語」**（○○には対象言語名が入る）

### ターゲットユーザー
- ○○語を学びたい日本人
- 通勤・通学・運転中の「ながら学習」を求める人
- 旅行・駐在・国際結婚など実用目的の学習者

### 中核価値
1. **聞き流しで耳を鍛える** — ネイティブ音声を反復、画面を見なくても学べる
2. **瞬間作文で口を慣らす** — 日本語からターゲット言語へ即座に変換する練習
3. **単語フラッシュカードで語彙を増やす** — 短時間で効率的に
4. **無料でも実用的なボリューム** — 課金前にしっかり試せる

## 2. アプリ構成（3つの学習モード）

### モード1: 会話モード（💬）

**目的**: 日本語を見て瞬時にターゲット言語へ変換する練習

**フロー**:
```
HOME → テーマ選択 → レベル選択 → 練習画面
                                    ↓
                              Phase A（問題表示）
                                    ↓ 「言語を表示」ボタン
                              Phase B（答え表示）
                                    ↓ 前/次ボタン
                              次の例題
```

**機能**:
- **翻訳方向トグル**: 日本語→ターゲット語 / ターゲット語→日本語 を切替可能
- **音声ボタン**: Phase A・Phase B 両方に「🔊 音声を再生」ボタン
- **自動再生なし**: ユーザーが明示的にボタン押下時のみ音声再生（ながら学習中の予期せぬ音を防ぐ）
- **前/次ボタン**: 例題間の移動
- **キーボード**: Space でリビール、←→ で例題移動（Web版）

### モード2: 聞き流しモード（🎧）

**目的**: ネイティブ音声を自動連続再生して耳を鍛える

**フロー**:
```
HOME → テーマ選択 → レベル選択（設定込み）→ 聞き流し画面
                                                ↓
                                          自動連続再生
                                          - 1段目言語
                                          - 0.4秒間
                                          - 2段目言語（リピート可）
                                          - 1.2秒間
                                          - 次例題
                                          - レベル終了で次レベル
                                          - テーマ終了で次テーマ
                                          - 無料層終了でループまたは停止
```

**機能**:
- **再生順序の選択**: 日本語→ターゲット語 / ターゲット語→日本語
- **ターゲット語の再生回数**: 1〜3回（学習対象を反復）
- **ノンストップトグル**: 全テーマ・全レベルを横断してループ再生
- **速度調整**: 0.8x / 1.0x / 1.2x / 1.5x の4段階
- **バックグラウンド再生**: 画面ロック・他アプリ起動中も継続
- **再生中の言語ハイライト**: アクティブな言語カードが色付き
- **前/次/再生・停止ボタン**

### モード3: 単語モード（📚）

**目的**: 分野別フラッシュカードで語彙を効率的に習得

**フロー**:
```
HOME → 分野選択（30カテゴリ）→ 方向選択 → フラッシュカード
                                ↓
                          ターゲット語 → 日本語 または
                          日本語 → ターゲット語
                                ↓
                          タップ/Space で表裏反転
                          前/次ボタン
                          🔀 シャッフルトグル
                          ⏱ 自動反転トグル
```

**機能**:
- **30分野のカテゴリ**（文法・機能語 10、ヒト・モノ 10、場所・社会 10）
- **両方向出題**: 訳語→日本語 / 日本語→訳語
- **3Dフリップアニメーション**: タップで表裏反転
- **シャッフルモード**: ランダム出題
- **自動反転モード**: 数秒で自動フリップ→次カードへ

## 3. 無料層 vs プレミアム

### 無料層（ストア初期公開バージョン）
- **会話モード**: 5テーマ × 3レベル × 20例題 = **300例題**
- **聞き流しモード**: 同上の音声（日本語・ターゲット語 各300ファイル）
- **単語モード**: 5分野 × 約20語 = **100語**

### プレミアム（将来課金）
- 全30テーマ × 3レベル = **1,800例題**
- 全30分野 × 約30語 = **約1,000語**
- 音声・単語訳全コンテンツ
- 中級・上級コンテンツの完全開放
- 価格目安: 月¥500 または 買い切り¥1,500

### Paywall 動作
- 無料層外のテーマ・レベル・分野を選ぶと **Paywall画面** に遷移
- 「🔒 プレミアム」表示で含まれる機能を訴求

```typescript
  jp: string;   // 日本語
  ne: string;   // ターゲット語（言語によって変える）

```

```typescript
interface Word {
  ja: string;   // 日本語訳
  ne: string;   // ターゲット語
}
```

### Theme Meta

```typescript
interface ThemeMeta {
  id: number;       // 1〜30
  name: string;     // テーマ名
```

## 10. 他言語版を作る手順（実装フロー）

### 想定: 同じコンセプトで「聞いて話せる英語」を作る

#### Phase 1: コンテンツ生成（数日〜1週間）

1. **テーマ・分野のリスト確認**（日本語版と同じ30テーマ × 30分野で良い、文化依存項目は調整）
2. **Anthropic APIで例文生成**:
   ```bash
   # 既存スクリプトを言語切替で再実行
   npm run generate-excel  # → 瞬間作文.xlsx (英語版)
   ```
   - プロンプトで「日本語↔英語」に変更
3. **単語データ作成**:
   ```bash
   npm run build-words-excel  # → 単語.xlsx (英語)
   ```
   - スクリプト内の単語リストを手動で英訳
4. **音声生成（Google Cloud TTS）**:
   ```bash
   npm run generate-audio-bulk      # 英語音声 → english/ フォルダ
   npm run generate-audio-bulk-ja   # 日本語音声 → japanese/ フォルダ
   ```
5. **単語辞書生成（Google翻訳）**:
   ```bash
   npm run build-vocab  # data/vocab.json
   ```

#### Phase 2: 開発（数日）

1. **新ディレクトリでExpoプロジェクト作成**:
   ```bash
   npx create-expo-app@latest expo-app --template blank-typescript
   ```
2. **既存コード移植**:
   - `src/` ディレクトリ全体をコピー
   - `data/` ディレクトリも雛形コピー
   - `assets/audio/` を新言語用に差し替え
3. **言語固有の調整**:
   - `src/theme.ts`: アクセント色を文化に合わせて変更
   - `src/screens/*`: 文字列「ネパール語」を「英語」に置換
   - `expo-app/app.json`:
     - `name`: 「聞いて話せる英語」
     - `bundleIdentifier` / `package`: `com.safa.english.jp` 等
     - `slug`: 別の slug
4. **データエクスポート**:
   ```bash
   npm run export-to-expo  # 新言語のExcel・音声からJSON生成
   ```

#### Phase 3: ビルド・配布（1-2週間）

1. **EAS で同じExpoアカウントに新プロジェクト追加**:
   ```bash
   cd expo-app
   eas init  # 新プロジェクト作成
   ```
2. **AAB ビルド & ストア提出**:
   - 日本語版と同じ流れ
   - Apple/Google Play Console で**新アプリ**として登録（パッケージ名が違うので別アプリ扱い）

#### 重要：「safa」シリーズとして展開

- ブランド名「safa」を統一して **アプリ名: 「聞いて話せる○○語」** 形式で展開
- Apple/Google Play の **デベロッパー名を統一**することで「safa シリーズ」としてユーザーが認識しやすい
- 各アプリのストアページ末尾に「同じ作者の他のアプリ」リンクを追加して相互流入を促す

## 11. 起動・運用の重要ポイント

### 無料層を絞る基準
- 無料層は「実用最低限」を確保する（5テーマ × 3レベル = 300例題は手応えあり）
- 単語は分野を絞って「文法・機能語」中心にする（文型の基礎を網羅）
- 無料層だけで2〜3週間遊べる量がベスト

### プレミアム転換の鍵
- 無料層を**使い切ってもらう**（途中でやめさせない）
- 「次のレベル」「次のテーマ」を頻繁に視覚化（ロックアイコンで誘惑）
- ユーザーが「もっと知りたい」と思った瞬間に解放できる体験

### ローンチ後の改善
- **Firebase Analytics** でモード別利用率を計測（どこで離脱するか）
- **Play Console のフィードバック**を月1回チェック
- 3ヶ月で1回のメジャーアップデート（新コンテンツ追加 + 機能改善）
