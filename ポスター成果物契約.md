### 1-3. 教材音声(`_正規化` を消費)
```
共通: <教材ルート無関係>/多言語教材/00_共通/音声/_正規化/<LANGDIR>/<NN>_<和名テーマ>/<MM>_<L>.mp3

```
- `<LANGDIR>` = 言語の大文字フォルダ: ja→`JA`, en→`EN`, bn→`BN`, vi→`VI`, zh→`ZH`, ne→`NE`(ko→`KO` 予定)。
- `<MM>` = カード通し番号(01..n、テーマ内)。`title_<L>.mp3` = テーマ見出し読み。
- **声 = 教材標準(edge-tts)**。ne=`ne-NP-HemkalaNeural`, ja=`ja-JP-NanamiNeural` 等。

- `_正規化` = ラウドネス/整形済み。**生(`音声/<LANGDIR>/...`)ではなく必ず `_正規化` を使う。**

## 3. コンシューマ・パイプライン(本リポジトリ)

### 3-1. ブリッジ `聞いて話せる日本語/scripts/_gen_poster.py`
契約パスを唯一の入力に、以下を生成(**生成物=アプリ実体**):
1. `expo-app/assets/poster/<key>/poster_<L>.png` … 広告なしポスターをコピー(マスター/再公開用に温存。Metroはバンドルしない)。
2. `expo-app/assets/poster/<key>/audio/<MM>_<L>.mp3`, `title_<L>.mp3` … 教材音声を **32k mono 24kHz + 末尾0.25s無音pad** へ再エンコード(`ffmpeg -ar 24000 -ac 1 -b:a 32k -map_metadata -1 -af apad=pad_dur=0.25`)。padは末尾切れ防止。
3. `expo-app/src/posterLessons.ts` … `id/title/titleL1/imageL1/titleAudio/posterW/posterH/cards[{i,box,ja,l1}]`。boxは**ポスター画像から実カード枠を検出**。
   - **解像度非依存であること**: `posterW/posterH` は実画像サイズから取得し、検出列座標 `COL` も画像幅に比例スケールする。**boxを検出した画像 = パックに積む画像 = posterW/posterH** の三者が同一画素空間である限り、アプリ実行時のオーバーレイ/拡大は正しくスケールする(アプリは座標系基準)。
   - ∴ 教材が2480→1240へ変わっても、再生成すれば自動追従(posterW/H=1240/1754, box=1240基準)。**画像と座標は必ずセットで再生成・再公開する。**

### 3-2. パック化 `聞いて話せる日本語/scripts/build-poster-pack.py`
`assets/poster` を言語別 STORED zip 化 → Release `packs-poster` へ。
- `poster-<L>.zip`: `<key>/audio/<MM>_<L>.mp3`(title含む) + `<key>/poster_<L>.png`。ja は音声のみ(画像なし=母語画像にtarget内包のため)。
- `poster-catalog.json`: `{ version, langs:{ L:{ url, bytes, version } } }`。
- STORED(無圧縮)= ローダのストリーミング展開前提(音声は既に32k、png圧縮済)。

### 3-3. アプリ消費
- `expo-app/src/posterPackLoader.ts`: `ensurePosterPack(lang)` で `poster-catalog.json`→zip取得・展開。`posterUri(key)` = `${documentDirectory}poster/${key}`。
- `posterLessons.ts` の値は**zip内エントリのキー文字列**(例 `family/audio/01_ja.mp3`)。`file://` 解決はローダ。
- DL対象 = ja(target音声) + 現在のL1(母語の音声+画像)。
- 再生 = ダブルバッファ(`PosterAudioScreen.tsx`)。キュー: title-L1→title-ja→card0-L1→card0-ja→…。`onLayout` 実測でcontain-fit配置。

## 4. 変更時の追従ルール(どれが何をトリガするか)

| 変わったもの | 必要な追従 | 再ビルド要否 |
|---|---|---|
| 音声だけ(声/読み修正、レイアウト不変) | `_gen_poster.py` 再生成 → `build-poster-pack.py --publish` | **不要**(差分DL) |
| ポスター画像のレイアウト/解像度 | `_gen_poster.py` 再生成(box再検出・posterW/H更新) → posterLessons.ts更新 → パック再公開 | **要**(posterLessons.ts はバンドル) |
| テーマ追加/キー確定 | `THEMES` と本表§2に追記 → 再生成 → 公開 | 要 |
| 新L1追加(例: ko) | プロデューサが ko ポスター+ko音声を契約パスに用意 → `_gen_poster.py` の L1集合に ko 追加 → 再生成 → 公開 | 要(posterLessons.ts に ko キー追加) |
